# AI Agents Context

This document stores the context and prompts used to generate the Tea4Life frontend.

## The Ultimate Prompt

Act as a Senior UI/UX Designer. Design a full-stack Bubble Tea E-commerce website named 'Tea4Life'.

1. Aesthetic & Theme:

- Visual Style: A seamless blend of Modern Minimalism and 16-bit Pixel Art.
- Color Palette: 'Nature's Breath'—utilizing a gradient of Forest Green, Sage Green, and soft Cream White.
- Icons: Custom pixel-art icons for Categories (Milktea, Fruit Tea, Toppings).

2. Core Components to Design:

- Hero Section: High-quality imagery of a fresh matcha latte surrounded by pixel-art leaves and nature elements.
- Product Grid: Cards must show: Name, Image, Base Price, and a 'Customize' button.
- Customization Modal (The 'Ordering' Logic):
  - A clean list of Product Options (Size, Sugar, Ice, Toppings) sorted by sortOrder.
  - Checkboxes for multi-select Toppings with extraPrice clearly visible (e.g., +5,000đ).
  - Radio buttons for single-select Options like Size or Sweetness levels.

3. Technical Requirements:

- UI Framework: Responsive layout suitable for React and Tailwind CSS.
- Vibe: Eco-friendly, organic, tech-forward (Microservices ready), yet nostalgic and cozy.
- Output: Create a high-fidelity visual representation of the Homepage and the Product Customization Modal.
